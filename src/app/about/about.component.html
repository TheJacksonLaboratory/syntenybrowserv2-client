What is JAX Synteny Browser?

The JAX Synteny Browser is an interactive, genome-agnostic, web-based application that allows researchers to select or highlight features based on annotated biological properties in reference and comparison genomes in the context of regions of conserved synteny (a condition in which shared ancestry is reflected as similar genome feature content and order along a chromosome in a different species) between the two genomes.

While it is genome-agnostic, the online JAX Synteny Browser is preloaded and configured for comparing mouse, rat and human genomes along with feature annotations for Gene Ontology (all species), Disease Ontology (human), and Mammalian Phenotype Ontology (mouse & rat). The application is open-source so if additional species or ontologies are of interest to you, assuming that you have access to the necessary source data (see the Data Prep and Config Files pages for more information), you can load a custom version of the database and API and view the custom set of data using a local version of the client application as well.
