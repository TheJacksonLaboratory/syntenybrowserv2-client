Tutorial

This tutorial describes how to set up a local instance of the JAX Synteny Browser and load some common data types.

Prerequisites

To follow the steps in this tutorial you will require the followng packages:

  • Docker(https://www.docker.com/community-edition): Docker is a program that lets you run "containers" hosting software and its dependencies
  • Python 3.x
  • synbrowser-manage:

Quickly viewing a dataset

The simplest way to get started is to open and view a dataset. The synbrowser-manage view command will automatically start a new instance if one isn't already running.
